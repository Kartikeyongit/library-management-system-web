# Library Management System (Web Version)

A web-based Library Management System built using HTML, CSS, and JavaScript.  
The application allows adding, viewing, searching, editing, issuing, returning, and deleting books.  
All book details are stored in the browser's LocalStorage for persistence.

## Features
- Add book
- Edit book
- Delete book
- Issue/Return book
- Search books by ID, title, or author
- Persistent data storage in LocalStorage
- Clean and interactive table-based UI

## How to Run
Open `index.html` in any modern web browser.
